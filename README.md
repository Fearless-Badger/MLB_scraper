Project which will scrape MLB roster information and integrate with a full web app


Requirements:

    Python 3.12.6
        -Flask
        -BeautifulSoup
        -requests
    node 23.1.0
        -npm
    

How to Run:

    cd into "Roster_App"
    run "pip install -r requirements.txt" terminal
    cd into "backend"
    run "python run.py"

    ctrl + click link in terminal

    ctrl + c to quit
